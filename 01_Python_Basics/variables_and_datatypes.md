<img width="1297" height="141" alt="image" src="https://github.com/user-attachments/assets/6bf24851-e0ab-49df-a202-09abb32d8ab5" />

```python
x = int(2.0)
y = float(12)
z = str(33)

print(x)
print(y)
print(z)

print(type(x))
print(type(y))
print(type(z))
```

Output:

```
2
12.0
33
<class 'int'>
<class 'float'>
<class 'str'>
```
